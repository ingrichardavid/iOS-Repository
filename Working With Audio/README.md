# Working With Audio.

# Requirements.
- Xcode 7.1
- Swift 2.1
- iOS 9

# Description.
Example of using audio files.

1.- Using constraints.

2.- Using the framework AVFoundation.

3.- Implementation of a progress bar.

4.- A small player with design functions: play, pause, stop and volume up and down.
